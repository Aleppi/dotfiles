# My dotfiles

## Installation

I use `yadm` for managing my dotfiles, so to install just do `yadm clone
https://github.com/Aleppi/dotfiles.git`. The bootstrap program may not work
properly at the moment.

## List of dependecies
* bspwm
* sxhkd
* polybar
* zsh
* Terminus
* Unifont
* Siji icons
* urxvt
* neovim
* ranger
* mpd
* ncmpcpp
* wal
* dunst
